# jhcasino

JHCasino portal site and playable game prototypes in a single repository.

## Structure

- `index.html`: Main portal
- `games/hi-low/index.html`: High & Low Dice
- `games/pachinko-slot/index.html`: Pachinko 3x3 Slot
- `games/dragon-tiger/index.html`: Dragon Tiger
- `games/crash/index.html`: Crash
- `games/plinko/index.html`: Plinko
- `games/plinko/physics.html`: Plinko Physics
- `games/roulette/index.html`: Roulette

## Hosting

This repository is ready for static hosting such as GitHub Pages.

For GitHub Pages:

1. Upload the contents of this folder to the repository root
2. Enable GitHub Pages in repository settings
3. Use the root branch as the publishing source
