# WARRIOR.IO v0.5

## 실행 방법
ES Module 구조라서 `index.html`을 더블클릭하면 브라우저 보안 정책 때문에 JS import가 막힐 수 있습니다.
아래 중 하나로 로컬 서버를 띄운 뒤 접속하세요.

```bash
python -m http.server 8080
```

그 다음 브라우저에서:

```text
http://localhost:8080
```

## 조작
- WASD / 방향키: 이동
- Shift + 이동: 달리기
- 마우스: 캐릭터가 바라보는 방향
- 마우스 클릭 유지: 마우스 위치로 이동
- Z: 공격
- X: 가드

## 변경 사항
- HTML/CSS/JS/에셋 구조 분리
- 런타임 검은색 배경 투명 처리 유지
- Shift 달리기 추가
- 마우스 방향 바라보기 및 클릭 이동 추가
- 적 공격 모션 잠금 처리 추가
- 플레이어 사망 중 추가 피격/히트 연출 방지
