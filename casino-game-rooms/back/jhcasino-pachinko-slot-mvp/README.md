# jhcasino-pachinko-slot-mvp

MVP prototype of a cartoon-style 3x3 pachinko slot game for JHCasino.

## Overview

This repository contains a lightweight static HTML prototype for a compact cartoon slot machine experience.

## Files

- `index.html`: Main playable prototype

## Deploy

This project is ready for static hosting such as GitHub Pages, Netlify, or Cloudflare Pages.

For GitHub Pages:

1. Create a repository named `jhcasino-pachinko-slot-mvp`
2. Upload the contents of this folder
3. Enable GitHub Pages in repository settings
4. Use the root branch as the publishing source
